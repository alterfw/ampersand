Ampersand, a micro framework for WordPress.

Ampersand is a micro framework for WordPress based on Slim that helps you quickly write websites and themes as real web applications. 

Route::get("/hello/:name", function ($name) {
  echo "Hello, ".$name;
});

Download & Install

To install Ampersand you'll need Composer dependency manager. Run this bash command in your theme's root folder to download and install the latest stable version into your theme's vendor/ directory. 

composer require alterfw/ampersand

Next, require the Composer autoloader into your functions.php. 

<?php
require 'vendor/autoload.php';

Features

Router

Ampersand provides a simple router built in the WordPress rewrite API that maps route callbacks to specific HTTP request methods and URIs.

HTTP

Ampersand has easy-to-use request and response abstractions that let you inspect and manipulate HTTP messages' method, status, URI, headers, cookies, and body.

Templates

Ampersand uses Twig as built-in as template engine, so you can create powerful templates using layouts, template blocks, etc.

Models (with hero)

Ampersand doesn't have native support to models but works well with Hero, a model-binding system to WordPress Post Types and Taxonomies.
